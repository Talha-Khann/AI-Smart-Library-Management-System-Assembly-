# AI Smart Library Management System (Assembly)



## Overview



This project is a smart library management system developed in Assembly language. It is built using JWasm (MASM-compatible assembler) along with the Irvine32 library.



The system runs in a console environment and demonstrates how low-level programming can be used to manage records, handle user input, and perform structured operations.



---



## Features



- User login system with username and password

- Add new books to the system

- Display available books

- Search for specific books

- Issue books to users

- Return books

- Update book records

- Delete books

- Fine calculation for late returns



---



## Technology Used



- Assembly Language (x86)

- JWasm / MASM

- Irvine32 Library



---



## How the System Works



The program starts with a login system. After successful login, the user is shown a menu with different options such as adding books, searching, issuing, and returning.



Each operation is handled using procedures and conditional logic. Data is stored and managed using memory structures defined in Assembly.



---



## How to Run



1. Install JWasm or MASM

2. Set up Irvine32 library properly

3. Assemble the file:

   jwasm main.asm

4. Link and run the program
